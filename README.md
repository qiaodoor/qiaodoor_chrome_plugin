qiaodoor_chrome_plugin
======================

chrome data fetch plugin

STEP 1: download the code and put it to local e.g.: d:/dev/qiaodoor_chrome_plugin

STEP 2: open chrome browser settings/extensions

STEP 3: 加载正在开发的程序

STEP 4: 登录www.qiaodoor.com并转到 Trades List，拷贝<Qiao Door Data Fetch Token>

STEP 5: 登录淘宝，并打开《已买到宝贝》，注意地址栏右角的QD图标，点击，在弹出框中粘贴<Qiao Door Data Fetch Token>，点击Upload

STEP 6: 回到http://www.qiaodoor.com/trades，并观察，会看到淘宝的购买数据已经抓到
